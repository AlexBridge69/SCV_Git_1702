# Работа с Git

## 1. Проверка наличия установленного Git

В терминале выполнить команду 
`git version`. Если Git установлен, появляется сообщение с информацией о версии программы. Иначе будет сообщение об ошибке.
## 2. Установка Git
Загружаем последнюю версию Git с [сайта] (https://git-scm.com/downloads)

## 3. Настройка Git
При первом использовании необходимо представиться. Для этого нужно ввести в терминале 2 команды:
```
git config --global user.name "Ваше имя"
git config --global user.email "Ваш e-mail"
```
